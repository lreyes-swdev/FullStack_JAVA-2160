Git Bash en VS Code
    VS Code >> Terminal >> New Terminal
        Abre el terminal integrado de Windows (Power Sheel)
    Para cambiar, en el panel que se abrió desplegar el botón "+" y seleccionar Git Bash
        Se abrirá una terminal Git Bash en el workspace de VSCode
    Para setear Git Bash como Terminal por defecto, desplegar el botón "+" y seleccionar Select Default Profile

#Consultar versión de git instalada
git --version

# Iniciar workspace
# - Se creará una carpeta .git en el workspace de VSCode
git init

# Agregar todos los archivos del workspace para control de versión
git add *

# Ver los estados de los archivos
git status

# Configurar identidad de Autor: user.name + user.email
git config --global user.email "lreyes.swdev@gmail.com"
git config --global user.name "lreyes-swdev"

# Confirmar los cambios de estado
git commit -m "Commit inicial | Clases 1~5"

# Confirmados de manera local, se puede hacer push hacia un repositorio GitHub
git remote add origin https://github.com/lreyes-swdev/FullStack_JAVA-2160.git
git branch -M main
git push -u origin main

# Luego, agregamos un nuevo archivo (por ejemplo index.html en la raíz de nuestro workspace)
git add .
git commit -m "agregamos index"
git push -u origin main
